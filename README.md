# Dota Stats Viewer

Live Demo: https://dota-stats-iey5npbdanv2o6pxzrdb5c.streamlit.app/

What it does:

- Takes a Steam profile link

- Downloads full match history using OpenDota API

- Calculates hero statistics

- Shows top heroes, winrates and games played

- Quick hero lookup
